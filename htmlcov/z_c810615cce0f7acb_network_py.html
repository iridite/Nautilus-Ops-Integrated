22 """ 

23 带重试机制的函数调用包装器 

24 

25 提供智能的重试策略，针对不同类型的网络错误使用不同的延迟时间。 

26 

27 Parameters 

28 ---------- 

29 func : Callable 

30 要执行的函数 

31 *args 

32 传递给函数的位置参数 

33 retries : int, optional 

34 最大重试次数，默认 3 

35 delay : float, optional 

36 基础延迟时间（秒），默认 5.0 

37 backoff_factor : float, optional 

38 指数退避因子，默认 2.0 

39 exceptions : Tuple[Type[Exception], ...], optional 

40 要捕获并重试的异常类型，默认 (ccxt.NetworkError, ccxt.ExchangeError) 

41 **kwargs 

42 传递给函数的关键字参数 

43 

44 Returns 

45 ------- 

46 Any 

47 函数执行结果 

48 

49 Raises 

50 ------ 

51 Exception 

52 当重试次数耗尽后，抛出最后一次的异常 

53 

54 Examples 

55 -------- 

56 >>> def fetch_data(): 

57 ... return exchange.fetch_ohlcv("BTC/USDT") 

58 >>> data = retry_fetch(fetch_data, retries=5, delay=2.0) 

59 

60 >>> # 直接调用带参数的函数 

61 >>> data = retry_fetch(exchange.fetch_ohlcv, "BTC/USDT", "1h") 

62 """ 

63 last_exception = None 

64 max_attempts = max(1, retries) # 至少执行一次 

65 

66 for attempt in range(1, max_attempts + 1): 

67 try: 

68 return func(*args, **kwargs) 

69 except exceptions as e: 

70 last_exception = e 

71 err_msg = str(e).lower() 

72 

73 # 计算延迟时间 

74 current_delay = delay 

75 

76 # 针对特定错误类型调整延迟 

77 if any(code in err_msg for code in ["418", "-1003", "rate limit", "too many requests"]): 

78 # API限流错误，使用更长的延迟 

79 current_delay = delay * backoff_factor * 2 

80 elif any(code in err_msg for code in ["timeout", "connection", "network"]): 

81 # 网络连接错误，使用指数退避 

82 current_delay = delay * (backoff_factor ** (attempt - 1)) 

83 

84 if attempt == max_attempts: 

85 # 最后一次尝试失败，抛出异常 

86 raise last_exception 

87 

88 # 等待后重试 

89 time.sleep(current_delay) 

90 

91 # 理论上不会到达这里，但为了类型检查器 

92 if last_exception: 

93 raise last_exception 

94 return None
